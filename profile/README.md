# WeblexAI Community

WeblexAI Community maintains the open source, self-hosted edition of WeblexAI.

The project is built for developers and teams who want to run their own translation infrastructure, manage their own providers, and keep deployment under their control.

## Projects

- `weblexai-community` - the self-hosted application
- `weblexai-community-docs` - documentation site, if separated later
